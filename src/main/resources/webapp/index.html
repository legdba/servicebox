HTTP services for testing of infra and containers.

GET /health/check

Returns "up".

GET /health/check/{percentage}

Returns "up" with {percentage} chance, or fail with an exception. {percentage} is a float number between 0 and 1. Usefull for health-checking scripts testing.

GET /echo/echo/{something}

Returns {something}.

GET /echo/echooo/{something}/{delayms}

Returns {something} after a {delayms} wait time.

GET /echo/ECHO/{something}

Returns {something} after CPU intensive math operations; good to cause CPU load.

GET /calc/add/{a}/{b}

Returns the integer result of {a}+{b}; Both {a} and {b} must be integers.

GET /calc/sum/{id}/{value}

Add {value} to the accumulator in backend at ID {id} and return accumulated value.

GET /leak/{size}

Leaks {size} bytes of data on Java heap and returns with a status of leaked and total leaked heap.

GET /leak/free

Frees all retained references causing the leak. Nex GC or Full-GC can reclaim associated heap.

GET /env/vars

Returns all system environmement variables.

GET /env/vars/{name}

Return the value of the system environmement variable {name}.

GET /env/hostname

Return the value InetAddress.getLocalHost().getHostName() which is usually good enough to get the hostname.
